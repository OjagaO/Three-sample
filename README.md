# 3Dモデルビューアー

## 機能概要

この3Dモデルビューアーは、gltfまたはglbファイルをドラッグ＆ドロップすることで、その3Dモデルを画面上に表示することができます。

## 使い方

1. **ファイルのドラッグ＆ドロップ**
   - gltfファイルの場合：gltfファイル、関連するbinファイル、およびtextureフォルダを一緒にドラッグ＆ドロップしてください。
   - glbファイルの場合：単一のglbファイルをドラッグ＆ドロップしてください。

2. **モデルの操作**
   - モデルはマウスで操作可能です。モデルを回転させたり、ズームイン・アウトしたりすることができます。

## 注意事項

- gltfファイルを使用する場合、関連ファイルがすべて含まれていることを確認してください。
- 一部の複雑なモデルや大きなファイルサイズのモデルは、正しく表示されない場合があります。
